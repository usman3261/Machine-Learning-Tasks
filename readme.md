
This repository contains a series of technical tasks focused on Exploratory Data Analysis (EDA) and predictive modeling. Each task is designed to demonstrate a specific stage of the machine learning lifecycle, from data quality auditing to deploying diagnostic AI pipelines.

---

## 🚀 Project 1: Iris Dataset Exploratory Analysis
**Focus:** High-Dimensional Data Visualization
* **Objective:** Identify physical traits that distinguish three species of Iris flowers.
* **Technical Highlights**: 
    * Multi-dimensional pair plots to identify linear separability.
    * Statistical correlation mapping (Pearson Matrix) to identify redundant features.
    * KDE distribution analysis to understand feature variance.

## 🏠 Project 2: House Price Prediction Pipeline
**Focus:** Regression Analysis & Feature Scaling
* **Objective:** Predict continuous property values based on physical house attributes.
* **Technical Highlights**:
    * Automated data quality audits for large datasets.
    * Implementation of `StandardScaler` to normalize features with different units.
    * Evaluation using Mean Absolute Error (MAE) and R-squared metrics.

## 🩺 Project 3: Clinical Heart Disease Diagnostic System
**Focus:** Binary Classification & Diagnostic Safety
* **Objective:** Predict the presence of heart disease using patient clinical data.
* **Technical Highlights**:
    * Stratified data splitting to maintain clinical case ratios in testing.
    * Optimized Random Forest Classifier for non-linear medical data.
    * Diagnostic Confusion Matrix visualization to assess model safety (False Positives/Negatives).
